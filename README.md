## Install

Clone 
```sh
$ git clone git@github.com:axlyody/kamuskata.git
```
Ambil vendor
```sh
$ composer update
```
Sesuaikan database kamu di file .env

Selanjutnya memasukan table dengan migrate
```sh
$ php artisan migrate
```
